# 浙超分组方案 — 数学建模竞赛 Spec

## 项目概述

**选题：B题 —— "浙超"分组方案**

2025年浙江省市县联赛，64支参赛队伍（11个市级 + 53个县级），分为16个小组（每组4队），进行单循环小组赛 + 5轮淘汰赛，决出冠军。

## 竞赛约束

- 截止时间：2026年5月18日 12:00（电子版）
- 论文格式：PDF，正文≤25页，附录含所有源码
- 允许使用AI工具，需在论文中标注并在支撑材料中提供使用说明

## 四个子任务

### Task 1：生成符合约束的分组方案（核心）

**硬约束：**
1. 11个市级队必须分配在不同小组（每组最多1个市级队）
2. 市级队不能与该市代管的县级队同组
3. 同一市代管的县级队尽量不同组（软约束，需优化）

**软目标：**
- 最小化同市县级队被分到同组的次数
- 组间实力均衡（如果有实力数据的话，可用GDP/人口等代理指标）
- 分组方案的多样性和可解释性

**方法论：**
1. **约束满足问题（CSP）建模**：用 Python + OR-Tools / Z3 求解
2. **贪心 + 随机化**：先分配市级队（确定性），再用带约束的随机分配县级队
3. **优化目标**：最小化违反软约束的次数，可用模拟退火/遗传算法做多目标优化
4. **枚举所有合法方案**：剪枝搜索，输出多个可行方案供对比

**输出：**
- 若干合法分组方案表格
- 每个方案的约束满足度量化指标
- 方案间多维度对比分析

### Task 2：抽签方案设计

**目标：** 设计一种公平、透明的抽签流程，使得抽签结果满足Task 1的所有约束。

**方法：**
1. 分层抽签法：
   - 第一层：11个市级队抽签确定各自小组（11个不同的组）
   - 第二层：每个市的县级队在"允许的组"中依次抽签
   - "允许的组" = 所有组 - 该市市级队所在组 - 已被同市其他县级队占据的组
2. 概率分析：计算各队落入各组的概率分布，证明均匀性
3. 模拟大量次抽签，统计结果分布，验证公平性

**输出：**
- 抽签流程的文字描述 + 流程图
- 概率分析和公平性证明
- 模拟抽签代码 + 统计结果

### Task 3：比赛地点选择

**目标：** 选择8个市/县作为小组赛举办地，每个地点承办2个小组。

**约束与优化目标：**
- 公平性：各队到比赛地的旅行距离尽量均衡
- 影响力：覆盖不同地区，扩大赛事影响
- 便利性：考虑交通基础设施（高铁、机场）

**方法：**
1. 收集浙江省64个市/县的地理坐标
2. 用聚类算法（K-means, K=8）初步划分赛区
3. 结合分组结果，优化分配使总旅行距离最小
4. 多目标优化：公平性 × 影响力 × 便利性

**输出：**
- 8个比赛地点及对应的小组分配
- 地图可视化（matplotlib / folium）
- 旅行距离分析

### Task 4：赛制建议

**目标：** 基于定量分析，提出赛制改进建议。

**分析方向：**
1. 当前赛制（16组×4队 → 32强淘汰赛）的公平性分析
2. 备选赛制对比：
   - 瑞士轮制（Swiss System）
   - 双败淘汰制（Double Elimination）
   - 更多/更少小组的方案
3. 蒙特卡洛模拟：不同赛制下各队夺冠概率分布
4. 竞技水平展示度：弱队是否能打足够多的比赛

**输出：**
- 多种赛制的对比表格
- 模拟结果和统计分析
- 建议及理由
